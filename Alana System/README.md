# Alana System


o projeto nasce da necessidade de tratar dados massivos do meu sistema de analise quantitatitva e criar o modelo responsavel pela analise dos dados.
efetivado o sistema, posso retirar a analise binaria que vem sendo feita ja ha algum tempo pra que eu possa entender de forma humana.
facilitando asssim toda uma analise procedural "por debaixos dos panos" me entregando uma analise mais detalhada e menos enviesada.



## Instalação

Instale as dependências usando o pip:

```bash
pip install 
```
